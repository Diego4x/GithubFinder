# GithubFinder
Projeto criado com a api do github

# introdução 
Com esse projeto simples um dev júnior pode aprender a fazer requisiçoes a api

```
//Faz a chamada a api

  then(res => res.json())

```

Com esse pequeno código o deve terá acesso a api deseja e depois e só usar como quiser
